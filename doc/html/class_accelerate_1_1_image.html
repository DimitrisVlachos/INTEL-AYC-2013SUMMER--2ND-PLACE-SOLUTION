Detailed Description

The Image class holds the data of the image.

To instantiate an image from a bitmap, you can use the static function:
Image::create_image_from_bitmap(const std::string file_name)

Returns the pixel at the position [i,j] in the image.

This function helps you to retrieve the value of a single pixel in the image. As the image is represented by a single dimension array, to retrieve the pixel at the position [i,j] you must compute i*image_width + j.
